Projetos

SIRAAS - SISTEMA DE REGISTRO DE ATESTADOS E AVALIAÇÃO SCRUM

O projeto SIRASS é um sistema web voltado para o gerenciamento de atestados médicos e a avaliação de equipes Scrum, desenvolvido para atender instituições de ensino e pequenas empresas. Minha principal contribuição no projeto foi a criação da página de gerenciamento de atestados, tanto para alunos quanto para docentes. Atuei principalmente no backend, utilizando HTML, Python e o microframework Flask para desenvolver a tabela de atestados e implementar o sistema de filtro de pesquisa.

Sphera

Projeto acadêmico desenvolvido em equipe na FATEC São José dos Campos, utilizando Java e princípios do Scrum. A aplicação tem como objetivo apoiar empresas na gestão, acompanhamento e análise de PDIs, fornecendo relatórios e indicadores para gestores e equipes de RH. Atuei na integração do banco de dados com o sistema em Java, além de contribuir na implementação de funcionalidades-chave e revisão do código em equipe.
